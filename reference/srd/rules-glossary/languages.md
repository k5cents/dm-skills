# Languages

Characters and creatures communicate in languages. A character knows Common plus two languages chosen from the Standard Languages table or granted by their background or class. Knowledge of a language means you can speak, read, and write it.

## Standard Languages

Standard languages are widespread in the worlds of D&D. Common originated in the planar metropolis of Sigil; the others originated with prominent species and spread widely.

| d12  | Language             | Origin       |
|:----:|----------------------|--------------|
|  —   | Common               | Sigil        |
|  1   | Common Sign Language | Sigil        |
|  2   | Draconic             | Dragons      |
| 3–4  | Dwarvish             | Dwarves      |
| 5–6  | Elvish               | Elves        |
|  7   | Giant                | Giants       |
|  8   | Gnomish              | Gnomes       |
|  9   | Goblin               | Goblinoids   |
| 10–11| Halfling             | Halflings    |
|  12  | Orc                  | Orcs         |

## Rare Languages

Rare languages are either secret or derived from other planes of existence and thus less widespread on the Material Plane. Some class features or other rules let a character learn a rare language.

| Language      | Origin                   |
|---------------|--------------------------|
| Abyssal       | Demons of the Abyss      |
| Celestial     | Celestials               |
| Deep Speech   | Aberrations              |
| Druidic       | Druidic circles          |
| Infernal      | Devils of the Nine Hells |
| Primordial*   | Elementals               |
| Sylvan        | The Feywild              |
| Thieves' Cant | Various criminal guilds  |
| Undercommon   | The Underdark            |

*Primordial includes the Aquan, Auran, Ignan, and Terran dialects. Creatures that know one dialect can communicate with those that know a different one.
